Buscador de Usados por Localização

Gere links prontos para pesquisar produtos usados no Facebook Marketplace, OLX e Mercado Livre usando sua localização atual ou um bairro/cidade manual.

Este projeto é 100% HTML, CSS e JavaScript. Ideal para publicar no GitHub Pages.

Resumo

Como funciona

  • O app usa o GPS do navegador ou uma localização digitada por você.
  • Depois monta links prontos para abrir nas plataformas.
  • Os sites podem exigir login ou ajuste manual de região.

Limitações reais

  • Este projeto não faz scraping e não coleta anúncios automaticamente.
  • Raio, preço e qualidade funcionam como apoio visual e contexto da busca.
  • Cada plataforma tem regras próprias e pode ignorar parte dos parâmetros.
Projeto estático pronto para GitHub Pages.
